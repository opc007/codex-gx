# 社区公约

## 我们的承诺

为了营造一个开放和友善的环境，我们作为贡献者和维护者承诺：无论年龄、体型、残疾、族裔、性别认同和表达、经验水平、国籍、个人形象、种族、宗教或性取向如何，参与我们项目和社区的每个人都不会受到骚扰。

## 我们的标准

有助于创造积极环境的行为包括：

- 🤝 使用友好和包容的语言
- 🎯 尊重不同的观点和经验
- 💡 优雅地接受建设性批评
- 🌟 关注对社区最有利的事情
- 💬 对其他社区成员表示同理心

不可接受的行为包括：

- ❌ 使用性别化的语言或图像，以及不受欢迎的性关注或性挑逗
- ❌ 侮辱/贬损的评论、人身或政治攻击
- ❌ 公开或私下骚扰
- ❌ 未经明确许可，发布他人的私人信息（如物理地址或电子邮箱）
- ❌ 在专业环境中可被合理认为不适当的其他行为

## 我们的责任

项目维护者有责任澄清可接受行为的标准，并期望对这些不可接受行为采取适当和公平的纠正措施。

项目维护者有权和责任删除、编辑或拒绝与本行为准则不符的评论、提交、代码、wiki 编辑、issue 和其他贡献，或暂时或永久地禁止任何他们认为不适当、威胁性、冒犯性或有害的贡献者。

## 适用范围

本行为准则适用于**所有** AgentShell 社区场合，包括但不限于：
- GitHub Issues / Pull Requests / Discussions
- 微信群 / 公众号
- 线下 meetup
- 项目代表发言时

## 举报

🚨 **遇到违反行为准则的情况**，请通过以下方式举报：

- **GitHub**：[私信 main maintainer](https://github.com/ahs)
- **邮箱**：`conduct@agentshell.dev`（即将开通）
- **微信群**：联系群主

**所有举报都会被及时审查和调查**。项目团队有义务为举报者保密。

## 归属

本行为准则改编自 [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)，
可在 [https://www.contributor-covenant.org](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) 查看完整文本。

社区影响指南可在 [Mozilla 行为准则执行阶梯](https://github.com/mozilla/diversity) 找到。
